# SipACL

```bash
git clone https://github.com/<your-org>/SipACL
cd SipACL
```

**Install** (from repo root):

```bash
pip install -r reqs.txt
```

**Run the policy** (from repo root). Use either script—both do the same sweep; pick whichever works in your shell:

- **`.sh`** (bash, Git Bash, WSL, or Linux/macOS):  
  `./Work/run_resample_sweep.sh`
- **`.ps1`** (Windows PowerShell):  
  `.\Work\run_resample_sweep.ps1`

On Windows, the `.sh` often works too (e.g. if you have Git for Windows or WSL). If one fails, try the other.
